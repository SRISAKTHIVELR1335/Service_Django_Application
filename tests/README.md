# tests/ – Automated Testing

This folder organizes automated tests for the entire NIRIX Diagnostics platform.

Structure:
- backend/unit/             → Fast unit tests for backend modules
- backend/integration/      → Integration tests hitting the API & DB
- windows-client/unit/      → Unit tests for Windows client code
- android-client/e2e/       → End-to-end tests for the React Native app
- system/                   → Cross-component system tests
